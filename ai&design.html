RoomieFunds Application

RoomieFunds is a mobile app that simplifies financial management for roommates. It tracks group savings goals, auto-splits expenses, and offers personalized financial tips. The app features an intuitive dashboard with savings progress and expense visualizations to promote transparency. Designed with user-centered research, RoomieFunds minimizes budgeting friction and fosters better financial habits among roommates.


Problem Statement: Roommates often struggle with managing shared expenses and savings, leading to stress and confusion. There is an opportunity to simplify this process through automatic cost-splitting, savings goal tracking, and personalized financial tips.


Persona Using User Persona to create a persona by descriping the application.


How Might We's

1. How might we help Olivia and her roommates manage shared expenses more easily and transparently?

2. How might we reduce the financial stress and social friction caused by missed payments and disorganized expense tracking?

3. How might we encourage Olivia to save more consistently despite the unpredictability of shared expenses?

4. How might we provide budgeting and financial guidance that integrates seamlessly into Olivia's daily life and household dynamics?


Translating How Might We's to User Stories

1. As a user, I want a clear and simple way to manage and track shared expenses with my roommates, so that I can easily see who owes what and ensure payments are made on time.

2. As a user, I want to receive automatic reminders and notifications for shared expenses, so that I can avoid late payments and reduce financial tension with my roommates.

3. As a user, I want a system that helps me save consistently even when unexpected expenses arise, so that I can achieve my financial goals without stress.

4. As a user, I want budgeting tips and guidance tailored to my living situation, so that I can better manage my money and improve my financial literacy.


Wireframe

Using ChatGPT, I explored more ideas for a product (mobile application) that could address these user needs and stories. I wanted to expand more on a couple of ideas mentioned here: Group Savings Goals & Incentives and Expense and Savings Visualization. I wanted to know more about what the ideas would actually meand and how it would be translated to application features. Using these features, I used ChatGPT to further descrie how the features would look in an interface, the elements that would contain. From this information I was able to use the Wireframe Designer plugin on Figma to to create wireframes.
